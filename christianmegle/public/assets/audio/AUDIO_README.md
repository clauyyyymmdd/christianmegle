# Audio Assets Required

The following audio files need to be added to this directory:

## organ-swell.mp3
- Duration: 6-8 seconds
- Description: A majestic organ chord swell, typically used for absolution
- Suggested sources: Free church organ samples from freesound.org

## sanctus-bells.mp3
- Duration: 2-3 seconds
- Description: Traditional sanctus/altar bells ringing
- Suggested sources: Church bell recordings from freesound.org

## ambient-silence.mp3 (optional)
- Duration: 30+ seconds (will be looped)
- Description: Quiet ambient church atmosphere - distant echoes, subtle reverb
- Suggested sources: Cathedral ambience recordings from freesound.org
